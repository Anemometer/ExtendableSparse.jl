ExtendableSparse.SparseMatrixExtension — Type.
mutable struct SparseMatrixExtension{Tv, Ti<:Integer} <: SparseArrays.AbstractSparseArray{Tv,Ti<:Integer,2}

Struct to hold extension data of sparse matrix.

  • m::Integer

    Number of rows

  • n::Integer

    Number of columns

  • nnz::Integer

    Number of nonzeros

  • colptr::Array{Ti,1} where Ti<:Integer

    Linked list of column entries. Initial length is n, it grows with each new entry.

    colptr[index] contains the next index in the list or zero, terminating the list which starts at index 1<=j<=n for each column j.

  • rowval::Array{Ti,1} where Ti<:Integer

    Row numbers. For each index it contains the zero (initial state) or the row numbers corresponding to the column entry list in colptr.

    Initial length is n, it grows with each new entry.

  • nzval::Array{Tv,1} where Tv

    Nonzero entry values correspondin to each pair (colptr[index],rowval[index])

    Initial length is n, it grows with each new entry.

ExtendableSparse.ExtendableSparseMatrix — Type.
mutable struct ExtendableSparseMatrix{Tv, Ti<:Integer} <: SparseArrays.AbstractSparseArray{Tv,Ti<:Integer,2}

Extendable sparse matrix. A nonzero entry of this matrix is contained either in cscmatrix, or in extmatrix, never in both.

  • cscmatrix::SparseArrays.SparseMatrixCSC

    Final matrix data

  • extmatrix::SparseMatrixExtension

    Intermediate structure holding data of extension
